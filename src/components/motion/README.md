# Motion Animation

### package: @vueuse/motion

### Usage:

```typescript
import { Motion } from "@/plugins/motion";
```

```html
<motion>
  <h1>Hello World!</h1>
</motion>
```

### Options:

```typescript
interface Props {
  from?: any;
  to?: any;

  visible?: boolean;
  visibleOnce?: boolean;

  transition?: any;
  delay?: number;
  duration?: number;
  levitate?: boolean;
}
```
